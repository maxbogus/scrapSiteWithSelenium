# scrapSiteWithSelenium
## About
Project to parse Nasa Gov main page with Selenium and save img and meta data to parsed Data folder.

## Install
* Install virtual env
* Activate virtual env: $ source <env_name>/bin/activate
* Install project requirements: $ pip install -r path/to/requirements.txt
* download geckodriver
* place in current folder

## Run
* $ python scrapSite.py
* Check parsedData folder for images and text files.
